# NGOBRAS Project

This is the main project directory for NGOBRAS application.

## Directory Structure

```
project-root/
  ├── src/               # Source files
  │   ├── css/          # CSS stylesheets
  │   ├── js/           # JavaScript files
  │   ├── images/       # Image assets
  │   │   └── icons/    # Icon files
  │   ├── ngobras.html  # Ngobras main page
  │   └── index.html    # Main entry point
  └── dist/             # Distribution files
```
